Este proyecto implementa un sistema de creación de personajes para un juego de fantasía, utilizando patrones de diseño de software para una arquitectura flexible y extensible. El programa permite a los usuarios seleccionar una raza y, a partir de ella, generar diferentes partes del personaje (cuerpo, arma, armadura, etc.) de manera consistente.

El objetivo principal es demostrar la aplicación de los patrones Singleton y Abstract Factory para mantener la coherencia y evitar la creación de objetos de razas mixtas.
